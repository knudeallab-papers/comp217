----- How to use CVSManager -----

0. 프로그램 실행 환경
 * 프로그램의 실행을 위해 InteliJ, eclipse 등의 JAVA IDE 가 필요합니다.

1. 판매 기능 사용법 (메인 화면)
 * 메인 화면에서 판매할 상품의 바코드를 입력 후 '검색' 을 누릅니다.
  - 등록된 상품의 바코드만 입력 가능합니다.
 * 판매할 개수를 입력하고 '추가' 를 누릅니다.
  - 판매할 개수는 자연수를 입력하여야 합니다.
  - 판매할 개수는 재고를 초과할 수 없으며, 초과시 재고수량이 표시됩니다.
 * 행사여부에 따라 상품의 가격이 계산되고, 총액이 표시됩니다.
 * 우측에서 결제수단을 선택 후 '판매' 를 누르면 판매가 완료됩니다.
 * 판매한 물품만큼 재고에서 빠지게 됩니다.

2. 시재 점검 사용법 (메인 화면 -> 시재 점검)
 * 보유한 현금을 지폐종류별로 입력한 후 '계산' 을 누릅니다.
 * 현금판매금액과의 차이를 확인할 수 있습니다.

3. 재고 관리 기능 사용법 (메인 화면 -> 재고관리)
 ㄱ. 물품 발주 기능 (메인 화면 -> 재고관리 -> 물품 발주)
    * 상단의 필터를 적용하여 발주가능 상품목록을 확인할 수 있습니다.
    * 발주를 원하는 상품을 선택하고 '추가' 를 눌러 추가합니다. 
    * 상품을 다 추가했으면 '발주' 를 눌러 발주를 진행합니다.
    * 새로운 유통기한을 가진 물품들이 재고에 추가되게 됩니다.
 ㄴ. 재고 현황 기능 (메인 화면 -> 재고관리 -> 재고 현황)
    * 상단의 필터를 적용하여 재고현황을 확인할 수 있습니다.
    * 같은 종류의 물품이라도, 유통기한이 다르면 각각 따로 표시됩니다.
 ㄷ. 폐기 목록 기능 (메인 화면 -> 재고관리 -> 폐기 목록)
    * 상단의 필터를 적용하여 폐기목록을 확인할 수 있습니다.
    * '삭제' 를 누르면 폐기를 원하는 물품이 폐기됩니다.
    * 폐기된 물품은 재고에서 빠지게 됩니다.

4. 출퇴근 등록 기능 사용법 (메인 화면 -> 출퇴근 등록)
  * 직원 ID를 입력한 후 출근시간과 퇴근시간을 입력합니다.
  * 입력 후 '입력' 버튼을 누르면 정상적으로 근무시간이 저장됩니다.

5. 임금 정산 기능 사용법 (메인 화면 -> 임금 정산)
  * 직원들의 정보와 임금, 퇴직금이 표시됩니다.
  * '갱신' 버튼을 누르면 출퇴근 등록에서 등록한 근무시간이 반영됩니다.
   -52주 이상 근무하면 직원코드 끝자리가 3으로 갱신되고 퇴직금이 계상됩니다.
  * 임금 지급을 완료 했다면 '지급 완료' 버튼을 누르면 임금이 정산됩니다.
  * '고용' 버튼과 '해고' 버튼으로 직원을 고용 및 해고 할 수 있습니다.
