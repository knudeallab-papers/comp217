로그인과 회원가입은 기존의 웹사이트와 비슷한 방식으로 진행

회원가입을 마치면 program 폴더에 사용자의 개인 폴더가 생성됨

개인 폴더 안에는 사용자의 옷들을 저장할수있는 cloth 폴더와 사용자가 달마다 입었던 옷을 확인할 수 있는 date 폴더 존재
각 폴더안에는 사용자의 옷들의 정보를 기록하는 1,txt,(모자), 2.txt(상의), 3.txt(하의), 4.txt(신발)
사용자가 달마다 입었던 옷들의 정보가 적혀있는 2022_해당 월.txt가 있음

프로그램사용시 여기에 있는 txt파일을 읽고 써서 활용

좌측하단의 오늘입을 옷 정하기 버튼을 클릭하면 원하는 옷 스타일 항목을 클릭가능
그리고 프로그램 실행하면서 받아온 날씨의 기온정보를 이용해서 각각 봄가을(쾌적할때), 여름(더울때), 겨울(추울때)의 3가지 기온으로 나누어
계절마다 다르게 입는 옷의 특징을 그날 기온과 함깨 반영 (여름에도 추울수 있음, 이럴때 기온정보가 만약 겨울을 가리킬만큼 춥다면 겨울옷이 추천됨)

해당 버튼은 여러번 클릭해서 새로운 옷 생성 가능

마음에 드는 옷의 조합이 생겨났으면 결정하기를 클릭
결정하기 클릭시에 해당옷의 정보를 오늘 날짜를 이용해서 2022_해당 월.txt의 일자에 기록

좌측의 새로운 옷 등록은 jfilechooser를 사용 등록하고자 하는 옷의 정보를 간략히 입력한뒤 옷 저장 가능

새로운옷 사러가기버튼을 누를시 무신사 웹사이트가 나타남

캘린더 다이어리 버튼 누를시 사용자가 그 전에 무엇을 입었는지 확인할수있음 현재 예시는 6월 8일까지 생성되어있음

개인설정에서는 사용자의 프로필 사진, 체형, 비밀번호, 이메일을 수정할 수 있음

프로그램 종료버튼을 누르면 System.exit(0);
