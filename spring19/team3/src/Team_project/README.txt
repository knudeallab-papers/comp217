-How to run code?
	->첫번째로 보이는 창의 메뉴바에서 "Recommanded"를 누르면 가장 많이 주문된 3가지 항목을 볼 수 있다.
	->코드를 실행하면 처음 보이는 선택지(beverage, snack, ramen)에서 한가지를 선택한다.
	->새로 나타나는 창의 왼쪽 하단에 텍스트 필드에 돈을 입력하고 엔터를 누른다.(ex.10000)
	->원하는 제품을 미리 입력된 돈의 범위 내에서 자유롭게 선택한다.
	->입력된 돈의 범위를 벗어나도 오류창이 나타난다.
	->원하는 제품을 모두 선택하였다면 get버튼을 눌러 거스름돈을 받는다.

-Error control
	->돈을 입력하지 않고 주문한다면 텍스트 필드에 오류가 나타난다.
	->각 제품은 10개씩 저장되어있고, 11개 이상 구매를 시도하면 오류창이 나타난다.
	->제품이 모두 소진되면 처음 창의 메뉴 바에 있는 Add_stock을 이용해 해당 카테고리의 재고를 채운다.
	->입력된 돈의 범위를 벗어나도 오류창이 나타난다.
